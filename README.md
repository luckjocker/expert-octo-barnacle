## 一、标题

Markdown 支持六级标题，使用 `#` 号表示，`#` 数量对应标题级别。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

## 二、段落与换行

- **段落**：两段文字之间用空行隔开即可形成新段落。
- **换行**：在行末加两个空格后再回车，或者使用 `<br>` 标签强制换行。

```markdown
第一行文字  
第二行文字（行末两个空格）

这是新段落。
```

**效果：**

第一行文字  
第二行文字（行末两个空格）

这是新段落。

---

## 三、分割线

使用三个或以上的 `-`、`*` 或 `_`（中间可有空格）来创建分割线。

```markdown
---
***
___
```

**效果：**

---

***

___

---

## 四、文字样式

| 样式        | 写法                  | 效果        |
| --------- | ------------------- | --------- |
| **粗体**    | `**粗体**` 或 `__粗体__` | **粗体**    |
| *斜体*      | `*斜体*` 或 `_斜体_`     | *斜体*      |
| ***粗斜体*** | `***粗斜体***`         | ***粗斜体*** |
| ~~删除线~~   | `~~删除线~~`           | ~~删除线~~   |
| `行内代码`    | `` `行内代码` ``        | `行内代码`    |
| ==高亮==    | `==高亮==`（部分编辑器支持）   | ==高亮==    |

## 五、列表

### 5.1 无序列表

使用 `-`、`*` 或 `+` 开头：

```markdown
- 第一项
- 第二项
  - 嵌套项 A
  - 嵌套项 B
- 第三项
```

**效果：**

- 第一项
- 第二项
  - 嵌套项 A
  - 嵌套项 B
- 第三项

### 5.2 有序列表

使用数字加 `.` 开头：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**效果：**

1. 第一步
2. 第二步
3. 第三步

### 5.3 任务列表

使用 `- [ ]`（未完成）和 `- [x]`（已完成）：

```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项
```

**效果：**

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项

## 六、引用

使用 `>` 开头，支持嵌套：

```markdown
> 这是一段引用
>
> > 这是嵌套引用
```

**效果：**

> 这是一段引用
> 
> > 这是嵌套引用

## 七、代码

### 7.1 行内代码

用反引号包裹：

```
使用 `print()` 函数输出
```

**效果：** 使用 `print()` 函数输出

### 7.2 代码块

用三个反引号包裹，可指定语言实现语法高亮：

    ```python
    def hello():
        print("Hello, Markdown!")
    ```

**效果：**

```python
def hello():
    print("Hello, Markdown!")
```

常用语言标识：`python`、`javascript`、`java`、`html`、`css`、`bash`、`json`、`sql` 等。

## 八、链接

### 8.1 行内链接

```markdown
[链接文字](URL)
```

**效果：** [百度](https://www.baidu.com)

### 8.2 引用式链接

```markdown
[链接文字][ref]

[ref]: https://www.baidu.com "鼠标悬停提示"
```

**效果：** [链接文字][ref]

[ref]: https://www.baidu.com "鼠标悬停提示"

### 8.3 自动链接

```markdown
<https://www.baidu.com>
```

**效果：** <https://www.baidu.com>

## 九、图片

### 9.1 基础图片

```markdown
![替代文字](图片URL)
```

**示例：**

```markdown
![示例图片](https://www.baidu.com/img/flexible/logo/pc/result.png)
```

### 9.2 带链接的图片

```markdown
[![替代文字](图片URL)](链接URL)
```

**示例：**

```markdown
[![示例图片](https://www.baidu.com/img/flexible/logo/pc/result.png)](https://www.baidu.com)[![示例图片](https://www.baidu.com/img/flexible/logo/pc/result.png)](https://www.baidu.com)
```

## 十、表格

### 10.1 基础表格

使用 `|` 分隔列，`---` 分隔表头和内容：

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
```

**效果：**

| 姓名  | 年龄  | 城市  |
| --- | --- | --- |
| 张三  | 25  | 北京  |
| 李四  | 30  | 上海  |

### 10.2 对齐方式

在 `---` 中添加 `:` 控制对齐：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   | 内容     | 内容   |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
|:--- |:----:| ---:|
| 内容  | 内容   | 内容  |

## 十一、脚注

```markdown
这是一段包含脚注的文字[^1]。

[^1]: 脚注内容写在这里。
```

**效果：**

这是一段包含脚注的文字[^1]。

[^1]: 脚注内容写在这里。

## 十二、HTML 标签

Markdown 支持内嵌 HTML 标签：

```html
<details>
<summary>点击展开详情</summary>

这里是隐藏的内容，点击 summary 即可展开查看。

</details>
```

**效果：**

<details>
<summary>点击展开详情</summary>

这里是隐藏的内容，点击 summary 即可展开查看。

</details>

其他常用 HTML 标签：`<kbd>键盘按键</kbd>`、`<sub>下标</sub>`、`<sup>上标</sup>`、`<center>居中</center>`。

## 十三、转义字符

在特殊字符前加 `\` 来显示字面量：

```markdown
\* 不是斜体 \*
\# 不是标题
\[ 不是链接
```

**效果：**

\* 不是斜体 \*
\# 不是标题
\[ 不是链接

**需要转义的字符：** `\` `` ` `` `*` `_` `{}` `[]` `()` `#` `+` `-` `.` `!` `|`
